## Basic Programming note

### A. JavaScript (JS)
#### #1. Variable
```js
// For global variables
var age = 10;
var isValid = true;
var studentName = "Rady"

// For block scope variables
let age = 18;
let isValid = true;
let studentName = "Rady"

// For block scope variables with cannot be reassigned
const SCHOOL_NAME = "PNC";
const PI = 3.14;
```
